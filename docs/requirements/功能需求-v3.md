## 📐 详细功能需求

### 功能 1: 搜索框

#### 设计要求
- **高度**: 约 60px
- **圆角**: 20px
- **背景**: 毛玻璃效果 (Qt 5 Shader)
- **边框**: 蓝紫蓝渐变 (from-blue-500/30 via-purple-500/30 to-blue-400/30)
- **图标**: 左侧搜索图标
- **占位符**: 动态变化根据当前分类
- **字号**: 18px, semibold

#### 占位符规则
| 分类 | 占位符 |
|------|--------|
| all | 聚焦搜索 |
| apps | 应用程序 |
| files | 文件 |

#### 交互
- **聚焦**: 自动搜索全部内容
- **输入**: 实时搜索 (debounced 300ms)
- **ESC**: 清空输入并隐藏结果

#### Qt 5 毛玻璃实现
```qml
Rectangle {
    // 内容容器
    color: "rgba(207, 207, 207, 0.9)"

    // 毛玻璃效果 (Qt 5)
    layer.enabled: true
    layer.effect: ShaderEffect {
        fragmentShader: "
            varying highp vec2 qt_TexCoord0;
            uniform sampler2D source;
            uniform lowp float delta;
            uniform highp vec2 resolution;
            void main() {
                lowp vec4 color = texture2D(source, qt_TexCoord0);
                lowp vec3 sum = vec3(0.0);
                lowp int i, j;
                for (i = -2; i <= 2; i++) {
                    for (j = -2; j <= 2; j++) {
                        sum += texture2D(source, qt_TexCoord0 + vec2(float(i), float(j)) * delta / resolution).rgb;
                    }
                }
                gl_FragColor = vec4(sum / 25.0, color.a);
            }
        "
        property real delta: 3.0
    }
}
```

## 🗂️ 数据模型

### SearchResult 结构

```cpp
struct SearchResult {
    QString id;                    // 唯一标识
    QString name;                  // 名称
    Type type;                     // app/file/image/document

    // 应用程序专用
    QString category;              // 应用分类 (如 "通讯", "开发")
    QString nameEn;                // 英文名
    QString description;           // 额外描述
    QString iconPath;             // 应用图标路径

    // 文件专用
    QString path;                 // 完整路径
    QString extension;            // 文件扩展名
    QString fileType;             // 文件类型 (document/image/video/music)
    qint64 size;                 // 文件大小 (字节)
    qint64 modifiedTime;         // 修改时间
};

enum Type {
    App,
    File,
    Image,
    Video,
    Music,
    Document
};
```

## 🎓 搜索逻辑

### 默认行为 (all)
```cpp
QList<SearchResult> search(const QString& query, Category category) {
    QList<SearchResult> results;

    if (category == All || category == Apps) {
        results += searchApps(query);
    }

    if (category == All || category == Files) {
        results += searchFiles(query);
    }

    return results;
}
```

## 🚀 Qt 5 毛玻璃实现方案

### 方案 1: ShaderEffect (推荐)
```qml
ShaderEffect {
    id: blur
    property real blurRadius: 20
    property variant source: parent.parent

    fragmentShader: "
        #version 440
        layout(location = 0) in vec2 qt_TexCoord0;
        layout(location = 0) out vec4 fragColor;
        layout(std140, binding = 0) uniform buf {
            mat4 qt_Matrix;
            float qt_Opacity;
            float blurRadius;
            float delta;
            vec2 resolution;
        };

        uniform sampler2D source;

        void main() {
            vec2 uv = qt_TexCoord0;
            vec4 color = vec4(0.0);
            float total = 0.0;
            float radius = blurRadius;

            for (float x = -radius; x <= radius; x += 1.0) {
                for (float y = -radius; y <= radius; y += 1.0) {
                    vec2 offset = vec2(x, y) * delta / resolution;
                    color += texture(source, uv + offset);
                    total += 1.0;
                }
            }

            fragColor = color / total * qt_Opacity;
        }
    "

    property real delta: 2.0
    property vec2 resolution: Qt.vector2d(width, height)
}
```

### 方案 2: 快速模糊 (性能优化)
```qml
// 使用降采样 + 上采样
ShaderEffect {
    // 第一次模糊 (降采样)
    blurMax: 64
    blur: 1.0
    transparentBorder: true
}
```

### 方案 3: 渐变透明度 (最简单)
```qml
Rectangle {
    property alias color: backgroundRect.color
    color: color

    // 模拟毛玻璃的半透明 + 渐变
    gradient: Gradient {
        GradientStop { position: 0.0; color: Qt.rgba(0,0,0,0.1) }
        GradientStop { position: 1.0; color: Qt.rgba(0,0,0,0.05) }
    }
}
```
