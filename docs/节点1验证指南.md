mkdir build && cd build

## 🔧 技术亮点

1. **单例模式** - Database, Settings, SingleInstance 都使用单例确保全局唯一
2. **本地通信** - 使用 QLocalSocket 实现进程间通信
3. **SQLite 集成** - 完整的数据库表结构和索引
4. **动态搜索过滤** - QML 端实时过滤模拟数据
5. **Qt 5/6 兼容** - CMake 自动检测版本

## ⚠️ 已知限制

1. **全局快捷键**
   - 当前实现为占位符
   - Linux 平台需要 X11/Wayland 特定代码
   - 可在阶段 4 完整实现

2. **毛玻璃效果**
   - 当前使用纯色背景占位
   - 阶段 3 将使用 ShaderEffect 实现

3. **编译环境**
   - 需要 Qt 5.12+ 或 Qt 6
   - macOS 上需要安装 Qt
