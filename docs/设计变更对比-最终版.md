## ⚙️ 技术实现规格

### Qt 5 毛玻璃实现

使用 `ShaderEffect` + 着色器代码（见 `功能需求-v3.md`）

**关键技术点**:
```qml
layer.enabled: true
layer.effect: ShaderEffect {
    fragmentShader: "..."  // GLSL 代码
}
```

### 渐变边框实现

采用双层容器结构（外层渐变边框 + 内层内容）

**关键技术点**:
```qml
// 外层: 渐变背景 + 负边距
// 内层: 内容 + 圆角
```

### 数据模型

```cpp
struct SearchResult {
    QString id;
    QString name;
    Type type;      // App | File | Image | Document

    // App 专用
    QString category;
    QString nameEn;
    QString description;
    QString iconPath;

    // File 专用
    QString path;
    QString extension;
    QString fileType;
    qint64 size;
    qint64 modifiedTime;
};
```
